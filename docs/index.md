## 📖 關於這個部落格

這個部落格使用 [MkDocs](https://www.mkdocs.org/) 和 [Material for MkDocs](https://squidfunk.github.io/mkdocs-material/) 建置，部署在 GitHub Pages 上。

### ✨ 特色功能

- 📱 響應式設計，支援手機、平板、電腦
- 🌓 深色/淺色模式切換
- 🔍 全文搜尋功能
- 📝 Markdown 語法支援
- 💻 程式碼高亮顯示
